## 二、极简版启动方案（3000-5000 元）

### 2.1 预算分配

| 项目 | 费用 | 说明 |
|------|------|------|
| **店铺保证金** | 1000-2000 元 | 个人店/个体户店，可退还 |

| **直播设备** | 500-1000 元 | 手机支架 + 补光灯 + 麦克风 |
| **流动资金** | 1000-1500 元 | 代发垫资周转 |
| **合计** | **3000-5000 元** | 最低启动 |

### 2.2 设备清单（极简版）

| 设备 | 推荐 | 预算 |
|------|------|------|
| **手机** | 现有 iPhone 8+ 或安卓旗舰 | 0 元（自有） |
| **支架** | 手机三脚架 | 50-80 元 |
| **灯光** | 环形补光灯（10 寸） | 80-150 元 |
| **麦克风** | 领夹麦克风 | 100-200 元 |
| **背景** | 干净墙面/背景布 | 50-100 元 |
| **合计** | | **280-530 元** |

### 2.3 平台选择建议

**推荐：小红书个人店**
```
✅ 保证金最低（1000 元）
✅ 审核相对宽松
✅ 适合新手起步
✅ 客单价较高
⚠️ 流量增长较慢
```

**备选：抖音个人店**
```
✅ 流量大
✅ 直播带货成熟
✅ 工具生态完善
⚠️ 竞争更激烈
⚠️ 需要更多精力运营
```

### 2.4 启动步骤

```
Step 1: 注册账号（1 天）
├─ 下载平台 APP
├─ 注册账号 + 实名认证
└─ 完善个人资料

Step 2: 开通店铺（1-2 天）
├─ 申请个人店
├─ 提交身份证信息
├─ 缴纳保证金 1000 元
└─ 等待审核通过

Step 3: 选品采购（2-3 天）
├─ 1688 选品（5-10 款）
├─ 采购样品（300-500 元）
├─ 收到后检查质量
└─ 确定 3-5 款主推产品

Step 4: 上架商品（1-2 天）
├─ 拍摄产品照片
├─ 编写商品描述
├─ 设置价格（加价 50-100%）
└─ 上架到店铺

Step 5: 内容创作（2-3 天）
├─ 拍摄 3-5 条短视频/笔记
├─ 撰写种草文案
├─ 发布内容测试流量
└─ 根据数据优化

Step 6: 开始直播（第 2 周开始）
├─ 布置简易直播间
├─ 准备直播脚本
├─ 试播 1-2 场测试
└─ 正式开播（每周 3-5 场）

Step 7: 订单处理（持续）
├─ 客户下单后 1688 采购
├─ 填写客户地址
├─ 回填物流单号
└─ 处理售后问题
```

## 七、常见问题解答

### Q1: 没有营业执照可以开始吗？

**A:** 可以。抖音和小红书都支持个人店，只需身份证即可。但个体户店可开通更多类目，建议启动后尽快办理。

### Q2: 启动资金不够怎么办？

**A:** 
- 从极简版开始（3000 元）
- 先兼职做，有收入后再全职
- 选择保证金低的平台（小红书个人店 1000 元）
- 设备先用现有的

### Q3: 多久能看到收益？

**A:** 
- 第 1 个月：学习期，可能亏损或微利
- 第 2-3 个月：成长期，月利润 2000-5000 元
- 第 4-6 个月：稳定期，月利润 5000-20000 元
- 个体差异大，取决于投入和执行力

### Q4: 需要囤货吗？

**A:** 不需要。无货源模式的核心就是不囤货。有订单后再从供应商处采购，由供应商直接发货。

### Q5: 一个人能做吗？

**A:** 可以。初期一个人完全可以操作：
- 选品上架：2-3 小时/天
- 内容创作：2-3 小时/天
- 直播：2-4 小时/场
- 订单处理：1-2 小时/天
- 合计：8-12 小时/天（兼职可缩减）

### Q6: 选择什么类目好？

**A:** 新手推荐：
- 家居日用（需求稳定）
- 美妆工具（利润高）
- 数码配件（标准化）
- 母婴小件（复购高）
避免：服装（退货率高）、食品（资质要求）、大件（运费高）
