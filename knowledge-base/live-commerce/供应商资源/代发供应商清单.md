## 三、供应商筛选标准

### 3.1 基础筛选指标

| 指标 | 合格标准 | 优秀标准 |
|------|---------|---------|
| **开店时长** | ≥1 年 | ≥3 年 |
| **描述评分** | ≥4.6 分 | ≥4.8 分 |
| **服务评分** | ≥4.6 分 | ≥4.8 分 |
| **物流评分** | ≥4.6 分 | ≥4.8 分 |
| **回头率** | ≥15% | ≥30% |
| **月成交** | ≥5 万元 | ≥20 万元 |
| **发货时效** | 48 小时 | 24 小时 |

### 3.2 服务筛选指标

```
✅ 支持一件代发
✅ 支持 7 天无理由退换
✅ 24 小时客服响应
✅ 提供数据包（图片/文案）
✅ 支持混批（多款混拿）
✅ 库存实时更新
✅ 自动同步订单（ERP 对接）
```

### 3.3 风险评估指标

```
⚠️ 警惕以下情况：
- 新店（<6 个月）且成交量低
- 评分低于 4.5 分
- 大量差评
- 客服响应慢
- 发货超时频繁
- 退换货推诿
```

## 四、供应商开发流程

### 4.1 开发步骤

```
Step 1: 平台搜索
├─ 搜索目标产品
├─ 筛选「一件代发」
└─ 初步筛选（评分/销量）

Step 2: 详细评估
├─ 查看供应商档案
├─ 查看产品评价
├─ 查看退换货政策
└─ 记录候选供应商

Step 3: 联系客服
├─ 咨询代发政策
├─ 询问价格阶梯
├─ 确认发货时效
├─ 确认退换货流程
└─ 索要数据包

Step 4: 采购样品
├─ 购买 3-5 家样品
├─ 对比产品质量
```

## 五、供应商管理

### 5.1 备份原则

```
每个产品至少 2-3 家供应商备份
主供应商：价格最优，合作稳定
备用供应商 A：价格略高，质量可靠
备用供应商 B：应急使用
```

### 5.2 定期评估

**每月评估指标：**
- 发货及时率
- 产品质量稳定性
- 售后处理效率
- 价格竞争力
- 客服响应速度

**评估结果处理：**
- 优秀：增加合作，争取更优价格
- 合格：保持合作，持续观察
- 不合格：减少合作，寻找替代

### 5.3 关系维护

```
✅ 定期下单，保持稳定合作
✅ 及时付款，建立信用
✅ 问题沟通，及时反馈
✅ 量大后谈更优价格
✅ 长期合作，建立信任
```

## 六、避坑指南

### 6.1 常见骗局

```
❌ 加盟费骗局
   - 以「免费代发」吸引，后收取加盟费
   - 对策：选择正规平台，不交加盟费

❌ 预充值骗局
   - 要求预充值才能代发
   - 对策：选择无需预充值的供应商

❌ 质量不符
   - 样品与大批货质量不一致
   - 对策：每次发货前要求验货

❌ 虚假库存
   - 显示有货，下单后说没货
   - 对策：选择库存实时同步的供应商

❌ 盗图侵权
   - 使用未授权图片
   - 对策：使用供应商授权的数据包
```

### 6.2 合同/协议要点

```
如量大需要签合同，注意：
✅ 明确产品质量标准
✅ 明确发货时效
✅ 明确退换货政策
✅ 明确价格调整机制
✅ 明确违约责任
✅ 明确结算方式
```
