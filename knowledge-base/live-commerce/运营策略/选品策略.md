## 一、选品核心原则

### 1.1 选品黄金法则

```
✅ 市场需求大 - 有足够多人想买
✅ 竞争程度适中 - 不是红海类目
✅ 利润空间足 - 毛利率≥30%
✅ 适合无货源 - 不易损坏、标准化
✅ 视觉效果好 - 容易展示和种草
✅ 有故事可讲 - 能写出打动人心的文案
```

### 1.2 新手选品避坑

```
❌ 避免大类目红海（如服装、美妆大牌）
❌ 避免季节性太强的产品
❌ 避免易碎易损坏产品
❌ 避免尺码复杂的产品
❌ 避免食品（资质要求高）
❌ 避免大件商品（运费高）
❌ 避免侵权产品（品牌仿冒）
```

## 四、产品筛选标准

### 4.1 量化评分表

| 维度 | 权重 | 评分标准（1-5 分） |
|------|------|------------------|
| **市场需求** | 25% | 5=搜索量 10 万+，3=5-10 万，1=<5 万 |
| **利润空间** | 25% | 5=毛利≥50%，3=30-50%，1=<30% |
| **竞争程度** | 20% | 5=竞争小，3=中等，1=竞争激烈 |
| **产品品质** | 15% | 5=优质，3=良好，1=一般 |
| **视觉呈现** | 10% | 5=易展示，3=一般，1=难展示 |
| **售后风险** | 5% | 5=风险低，3=中等，1=风险高 |

**综合得分 = 各维度得分 × 权重**

**决策标准：**
- ≥4.0 分：重点主推
- 3.5-4.0 分：正常上架
- 3.0-3.5 分：备选观察
- <3.0 分：放弃

### 4.2 利润计算公式

```
售价 = 进货价 + 运费 + 平台扣点 + 预期利润 + 风险预留

毛利率 = (售价 - 进货价 - 运费 - 平台扣点) ÷ 售价 × 100%

示例计算：
进货价：30 元
运费：5 元
平台扣点（5%）：约 3 元（按售价 60 元计算）
预期利润：20 元
风险预留：2 元
─────────────────
建议售价：60 元
毛利率 = (60 - 30 - 5 - 3) ÷ 60 = 36.7%
```

### 4.3 竞争程度评估

**低竞争（⭐⭐⭐⭐⭐）：**
- 搜索结果<1000 个
- 头部商家销量<1000 件
- 价格不透明
- 有差异化空间

**中竞争（⭐⭐⭐）：**
- 搜索结果 1000-5000 个
- 头部商家销量 1000-5000 件
- 价格相对透明
- 需要一定差异化

**高竞争（⭐）：**
- 搜索结果>5000 个
- 头部商家销量>5000 件
- 价格战激烈
- 难以差异化

## 七、定价策略

### 7.1 定价方法

**成本加成法：**
```
售价 = 成本 × (1 + 目标毛利率)

示例：
成本 = 进货价 30 + 运费 5 + 平台扣点 3 = 38 元
目标毛利率 = 40%
售价 = 38 × (1 + 40%) = 53.2 元 ≈ 59 元
```

**竞争定价法：**
```
1. 调研同类产品价格区间
2. 确定自己的定位（低价/中价/高价）
3. 根据定位设置价格

示例：
竞品价格：49-79 元
自己定位：中端
售价：59-69 元
```

**价值定价法：**
```
1. 分析产品独特价值
2. 评估用户支付意愿
3. 根据价值定价

适用于：有差异化、有品牌溢价的产品
```

### 7.2 价格心理学应用

```
✅ 尾数定价：59 元比 60 元更有吸引力
✅ 锚定效应：原价 99 元，现价 59 元
✅ 套餐定价：单买 59 元，2 件 99 元（省 19 元）
✅ 阶梯定价：1 件 59 元，3 件 159 元（单件 53 元）
✅ 会员价：设置会员专享价增加粘性
```

### 7.3 各平台价格策略

| 平台 | 价格区间 | 加价倍率 | 说明 |
|------|---------|---------|------|
| **抖音** | 50-200 元 | 2-3 倍 | 冲动消费，需留投流空间 |
| **小红书** | 100-500 元 | 2-4 倍 | 品质导向，接受高客单 |
| **拼多多** | 20-100 元 | 1.3-2 倍 | 价格敏感，薄利多销 |
| **淘宝** | 50-300 元 | 1.5-2.5 倍 | 价格透明，竞争激烈 |

## 九、常见选品误区

### 9.1 新手常见错误

```
❌ 凭个人喜好选品
   → 应该：以数据为导向，看市场需求

❌ 盲目跟风热销品
   → 应该：分析竞争程度，找差异化

❌ 只看利润不看销量
   → 应该：综合评估利润×销量

❌ 不采购样品直接上架
   → 应该：必须测试样品再决定

❌ 只找 1 家供应商
   → 应该：至少 2-3 家备份

❌ 忽视售后风险
   → 应该：评估退货率和售后成本
```

### 9.2 避坑建议

```
✅ 用数据说话，不要凭感觉
✅ 小步快跑，先测试再放大
✅ 多元化选品，不要押注单一产品
✅ 持续学习，关注市场变化
✅ 建立选品库，持续积累
```
