## 一、投流核心原则

### 1.1 投流黄金法则

```
✅ 先有自然流量再投流 - 验证产品转化能力
✅ 小步快跑 - 小额测试，数据好再放大
✅ ROI 导向 - 以投入产出比为决策依据
✅ 持续优化 - 根据数据调整投放策略
✅ 控制风险 - 设置预算上限，避免亏损
```

### 1.2 投流时机判断

**适合投流的情况：**
```
✅ 自然流量转化率≥3%
✅ 产品有稳定供应链
✅ 直播间承接能力充足
✅ 有足够测试预算（3000-5000 元）
✅ 团队有投流经验或学习能力
```

**不适合投流的情况：**
```
❌ 产品转化率未验证
❌ 供应链不稳定
❌ 直播间承接能力弱
❌ 预算不足（<1000 元）
❌ 完全不懂投流
```

### 1.3 投流预算规划

| 阶段 | 日预算 | 测试周期 | 目标 |
|------|--------|---------|------|
| **测试期** | 100-300 元 | 7-14 天 | 验证产品转化 |
| **成长期** | 300-1000 元 | 14-30 天 | 放大盈利模型 |
| **稳定期** | 1000-5000 元 + | 持续 | 规模化盈利 |

### 2.5 千川数据分析

**核心指标：**
```
【消耗】
- 日消耗金额
- 计划消耗进度

【转化数据】
- 进入直播间人数
- 商品点击次数
- 下单订单数
- 成交金额（GMV）

【效率指标】
- 点击率（CTR）：点击/曝光
- 转化率（CVR）：下单/点击
- 投入产出比（ROI）：GMV/消耗
- 单次转化成本（CPA）：消耗/订单数
```

**数据标准（参考）：**
```
【优秀】
- CTR≥3%
- CVR≥5%
- ROI≥3
- CPA<利润的 50%

【良好】
- CTR 2-3%
- CVR 3-5%
- ROI 2-3
- CPA<利润的 70%

【需优化】
- CTR<2%
- CVR<3%
- ROI<2
- CPA>利润
```

## 五、投流 ROI 计算

### 5.1 ROI 计算公式

```
ROI = 销售额（GMV）÷ 投放消耗

示例：
投放消耗：1000 元
销售额：3000 元
ROI = 3000 ÷ 1000 = 3
```

### 5.2 盈亏平衡点计算

```
盈亏平衡 ROI = 1 ÷ 毛利率

示例：
产品毛利率 = 40%
盈亏平衡 ROI = 1 ÷ 0.4 = 2.5

解读：
- ROI>2.5：盈利
- ROI=2.5：保本
- ROI<2.5：亏损
```

### 5.3 综合 ROI 计算

```
考虑所有成本：
综合成本 = 产品成本 + 运费 + 平台扣点 + 投流成本 + 人工成本

综合 ROI = 销售额 ÷ 综合成本

示例：
销售额：10000 元
产品成本：4000 元
运费：500 元
平台扣点：500 元（5%）
投流成本：2000 元
人工成本：1000 元
─────────────────
综合成本：8000 元
综合 ROI = 10000 ÷ 8000 = 1.25
```

✅ 定期检查消耗进度
```

### 6.2 止损策略

**计划止损：**
```
【止损条件】
- 消耗达预算 50%，ROI<1
- 消耗达预算 80%，ROI<1.5
- 连续 3 天 ROI<盈亏平衡点

【止损操作】
- 暂停计划
- 分析原因
- 优化后重启或放弃
```

**账户止损：**
```
【止损条件】
- 日亏损超过预算
- 连续 7 天整体亏损
- ROI 持续下降

【止损操作】
- 暂停所有计划
- 全面复盘
- 调整策略后重启
```

### 6.3 常见风险

**账户风险：**
```
❌ 违规素材导致封户
❌ 频繁修改计划影响学习
❌ 预算设置过高消耗过快
❌ 定向过窄跑不出去
```

**数据风险：**
```
❌ 数据延迟导致误判
❌ 归因周期影响 ROI 计算

## 九、投流常见问题

### 9.1 新手常见问题

**Q：多少预算适合开始投流？**
```
A：建议至少准备 3000-5000 元测试预算。
   单条计划 100-200 元，测试 10-20 条计划。
   测试期 7-14 天，找到盈利模型后再放大。
```

**Q：ROI 多少算合格？**
```
A：参考标准：
   - ROI<1：亏损，需要优化或停止
   - ROI 1-2：接近盈亏平衡，需要优化
   - ROI 2-3：盈利，可以放量
   - ROI>3：优秀，大力投放

   具体标准根据产品毛利率计算盈亏平衡点。
```

**Q：投流后自然流量会下降吗？**
```
A：正常情况下不会。
   投流和自然流量是独立的。
   但如果投流导致直播间数据（停留/转化）下降，
   可能间接影响自然流量推荐。
   
   建议：保持内容质量，投流只是放大。
```

### 9.2 进阶问题

**Q：如何降低获客成本？**
```
A：方法：
   1. 优化创意素材（提高 CTR）
   2. 优化人群定向（提高 CVR）
   3. 优化直播间承接（提高转化）
   4. 选择竞争较小的时段
   5. 持续测试和优化
```

**Q：投流计划跑不动怎么办？**
```
A：可能原因：
   1. 出价过低 → 适当提高出价
   2. 定向过窄 → 放宽定向
   3. 素材吸引力不足 → 优化素材
   4. 账户权重低 → 养户、增加消耗
   5. 竞争太激烈 → 避开高峰时段
```
