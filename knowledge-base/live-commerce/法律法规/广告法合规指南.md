# 广告法合规指南

> 📅 创建日期：2026-03-01  
> 🎯 目标：了解直播电商广告法合规要求，避免违规处罚  
> 📊 重要性：⭐⭐⭐⭐⭐（违规可能导致高额罚款、封号）

---

## 一、广告法核心要点

### 1.1 适用范围

```
直播带货属于商业广告活动，适用《中华人民共和国广告法》

适用场景：
✅ 直播中介绍产品
✅ 短视频推广产品
✅ 图文笔记推荐产品
✅ 直播间展示产品
✅ 口播推荐产品
```

### 1.2 主要法规

| 法规名称 | 实施时间 | 核心内容 |
|---------|---------|---------|
| **广告法** | 2015 年修订 | 广告活动基本规范 |
| **电子商务法** | 2019 年 | 电商经营者义务 |
| **网络直播营销管理办法** | 2021 年 | 直播营销专门规定 |
| **消费者权益保护法** | 2014 年修订 | 消费者权益保护 |
| **反不正当竞争法** | 2019 年修订 | 禁止虚假宣传 |

## 二、直播违规红线（绝对不能碰）

### 2.1 绝对化用语（违禁词）

```
❌ 国家级、最高级、最佳
❌ 第一、首选、顶级
❌ 唯一、独家、独家专利
❌ 100%、绝对、肯定
❌ 永久、永久有效
❌ 万能、全效、特效
❌ 根治、治愈、疗效
❌ 无副作用、零风险
```

**替代说法：**
```
✅ 优质、优秀、良好
✅ 领先、前沿、先进
✅ 多款、多种、多款可选
✅ 大多数、通常、一般
✅ 长期使用、持久
✅ 多功能、多效
✅ 改善、帮助、有助于
✅ 温和、安全、放心
```

### 2.2 虚假宣传

```
❌ 虚构产品功效
❌ 夸大产品效果
❌ 伪造用户评价
❌ 虚假销量数据
❌ 伪造检测报告
❌ 虚构原价/折扣
❌ 虚假产地信息
❌ 虚假认证资质
```

**合规做法：**
```
✅ 如实描述产品功能
✅ 展示真实用户反馈
✅ 提供真实检测报告
✅ 明码标价，真实折扣
✅ 如实标注产地
✅ 展示真实资质证书
```

### 2.3 医疗功效宣称

```
普通商品禁止宣称医疗功效：

❌ 治疗 XX 疾病
❌ 预防 XX 疾病
❌ 具有 XX 疗效
❌ XX 天见效
❌ 根治/治愈 XX
❌ 替代药物
❌ 医生推荐
❌ 医院同款
```

**特殊商品要求：**
```
药品/医疗器械：
- 需要药品/医疗器械广告审查批准文号
- 只能在指定平台发布
- 不得在直播中销售处方药

保健品：
- 需要保健食品批准文号（蓝帽子）
- 不得宣称治疗功效
- 需标注"本品不能代替药物"

化妆品：
- 不得宣称医疗功效
- 特殊化妆品需注册
- 不得明示或暗示医疗作用
```

## 三、价格合规

### 3.1 价格标示要求

```
必须明示：
✅ 销售价格
✅ 计价单位（元/件、元/套等）
✅ 促销活动的期限
✅ 促销活动的条件
✅ 运费标准
```

### 3.2 禁止的价格行为

```
❌ 虚构原价
   - 原价必须是本次促销前 7 日内最低成交价
   - 不能随意标注"原价"

❌ 虚假折扣
   - 折扣计算必须准确
   - 不能先涨价再打折

❌ 低价诱导
   - 不能用低价吸引，实际不销售
   - 不能设置不合理购买条件

❌ 价格欺诈
   - 不能低价标高价卖
   - 不能隐瞒价格附加条件
```

### 3.3 合规价格标示示例

```
✅ 正确示例：
"这款衣服日常售价 299 元，今天直播间特价 199 元"
（需确保 299 元是真实售价）

"限时优惠，今晚 8-10 点下单立减 50 元"
（明确期限和条件）

"包邮，偏远地区除外（新疆、西藏、内蒙古）"
（明确运费条件）

❌ 错误示例：
"原价 999 元，今天只要 99 元"
（原价从未销售过）

"全网最低价"
（无法证明）

"买一送一"
（实际送的是小样，未说明）
```

## 四、产品资质合规

### 4.1 必须展示的资质

| 产品类别 | 必备资质 | 展示要求 |
|---------|---------|---------|
| **食品** | 食品经营许可证、生产许可证 | 直播中展示或商品页公示 |
| **化妆品** | 化妆品备案/注册信息 | 商品页公示 |
| **保健品** | 保健食品批准文号 | 直播中展示 |
| **医疗器械** | 医疗器械注册证 | 商品页公示 |
| **3C 产品** | 3C 认证 | 商品页公示 |
| **进口商品** | 海关报关单、检疫证明 | 可应要求提供 |
| **品牌商品** | 品牌授权书 | 可应要求提供 |

### 4.2 资质查验要点

```
查验供应商资质：
□ 营业执照是否有效
□ 生产/经营许可证是否在有效期
□ 产品检测报告是否齐全
□ 品牌授权是否真实
□ 进口商品手续是否完备

自查要点：
□ 店铺资质是否公示
□ 商品资质是否齐全
□ 宣传内容是否有依据
□ 产品与描述是否一致
```

## 五、消费者权益保护

### 5.1 七天无理由退货

```
适用商品：
✅ 大部分网购商品
✅ 商品完好不影响二次销售

不适用商品：
❌ 定制商品
❌ 鲜活易腐商品
❌ 数字化商品（已下载）
❌ 报纸期刊
❌ 其他根据商品性质不宜退货的商品

注意事项：
- 需提前告知消费者
- 退货条件要明确
- 退货运费承担要说明
```

### 5.2 三包责任

```
包修、包换、包退：

三包期限：
- 一般商品：7 天退货、15 天换货
- 电子产品：按国家规定
- 服装鞋帽：按行业标准

三包范围：
- 质量问题：商家承担
- 人为损坏：消费者承担
- 运输损坏：商家承担（可向物流追偿）
```

### 5.3 信息披露义务

```
必须披露的信息：
✅ 经营者真实名称和地址
✅ 联系方式
✅ 商品真实信息
✅ 价格和运费
✅ 售后服务政策
✅ 退换货规则
```

## 六、直播行为规范

### 6.1 主播行为规范

```
禁止行为：
❌ 吸烟、饮酒（除非酒类专场）
❌ 说脏话、低俗内容
❌ 涉及政治敏感话题
❌ 泄露他人隐私
❌ 诱导站外交易
❌ 虚假互动（刷单刷量）
❌ 恶意诋毁竞品
❌ 诱导未成年人打赏
```

### 6.2 直播内容规范

```
必须遵守：
✅ 内容健康向上
✅ 尊重社会公德
✅ 不侵犯他人权益
✅ 不传播虚假信息
✅ 不涉及黄赌毒
✅ 不涉及封建迷信
✅ 不涉及恐怖暴力
```

### 6.3 互动行为规范

```
合规互动：
✅ 回答产品问题
✅ 介绍产品功能
✅ 分享使用体验
✅ 引导关注店铺

违规互动：
❌ 诱导刷礼物
❌ 诱导站外加微信
❌ 诱导私下交易
❌ 泄露客户信息
❌ 辱骂、攻击他人
```

## 七、违规处罚

### 7.1 行政处罚

| 违规类型 | 处罚措施 | 罚款金额 |
|---------|---------|---------|
| **虚假宣传** | 责令停止、消除影响 | 20 万 -100 万元 |
| **绝对化用语** | 责令停止、消除影响 | 20 万 -100 万元 |
| **价格欺诈** | 责令改正、没收违法所得 | 5 万 -50 万元 |
| **无资质经营** | 责令停止、没收违法所得 | 1 万 -3 万元 |
| **侵犯消费者权益** | 责令改正、赔偿 | 视情节而定 |

### 7.2 平台处罚

```
抖音/快手/淘宝等平台处罚：
- 警告
- 限流
- 下架商品
- 暂停直播权限
- 扣除保证金
- 封禁账号
- 列入黑名单
```

### 7.3 民事赔偿

```
消费者可要求：
- 退货退款
- 三倍赔偿（欺诈行为）
- 损失赔偿
- 精神损害赔偿（严重情况）
```

## 九、合规话术模板

### 9.1 产品介绍话术

```
✅ 推荐话术：
"这款产品我们亲测过，效果很不错"
"很多客户反馈说..."
"根据官方资料显示..."
"这款产品在同类产品中表现优秀"
"适合 XX 人群使用"

❌ 避免话术：
"这是最好的产品"
"100% 有效"
"治疗 XX 疾病"
"全网第一"
"不买就亏了"
```

### 9.2 价格说明话术

```
✅ 推荐话术：
"日常售价 XX 元，今天直播间特价 XX 元"
"限时优惠，仅限今晚"
"前 100 名下单再减 XX 元"
"包邮，偏远地区除外"

❌ 避免话术：
"原价 XX 元"（未真实销售过）
"全网最低价"（无法证明）
"亏本甩卖"（可能虚假）
"最后 X 件"（虚假库存）
```

### 9.3 售后说明话术

```
✅ 推荐话术：
"支持七天无理由退货"
"质量问题我们承担运费"
"有任何问题联系客服"
"我们承诺 XX 天内发货"

❌ 避免话术：
"售出概不退换"
"质量问题也不退"
"运费自理"（未提前说明）
"不保证时效"
```

## 十、风险应对

### 10.1 被投诉处理流程

```
1. 收到投诉 → 立即响应
2. 核实情况 → 收集证据
3. 沟通协调 → 寻求和解
4. 无法和解 → 平台介入/法律途径
5. 处理完毕 → 复盘改进
```

### 10.2 被处罚应对

```
1. 收到处罚通知 → 仔细阅读
2. 评估处罚合理性 → 收集证据
3. 如不合理 → 申诉
4. 如合理 → 接受处罚、整改
5. 整改完毕 → 申请恢复
6. 持续改进 → 避免再犯
```

### 10.3 证据保存

```
需要保存的证据：
- 直播录像（至少 3 年）
- 商品页面截图
- 交易记录
- 聊天记录
- 物流信息
- 售后处理记录
- 资质证明文件
```

## 十二、总结

### 12.1 合规核心原则

```
✅ 真实 - 不虚假宣传
✅ 准确 - 信息准确无误
✅ 完整 - 信息披露完整
✅ 及时 - 响应及时处理
✅ 诚信 - 诚信经营为本
```

### 12.2 合规价值

```
短期看：避免罚款、封号风险
中期看：建立品牌信誉
长期看：可持续发展基础
```
