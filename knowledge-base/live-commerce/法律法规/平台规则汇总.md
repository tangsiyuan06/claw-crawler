## 一、规则遵守核心原则

### 1.1 合规经营黄金法则

```
✅ 熟读平台规则：定期学习官方规则更新
✅ 不碰红线：不售假、不虚假宣传、不引导私下交易
✅ 如实描述：产品信息真实准确
✅ 优质服务：及时处理售后和投诉
✅ 关注更新：平台规则会不定期更新
```

### 1.2 违规后果

| 违规程度 | 处罚措施 | 影响 |
|---------|---------|------|
| **轻微违规** | 警告、下架商品 | 短期流量影响 |
| **一般违规** | 扣分、限制功能 | 7-30 天影响 |
| **严重违规** | 封店、冻结资金 | 长期或永久影响 |
| **违法行为** | 移交司法机关 | 法律责任 |

## 二、抖音直播规则

### 2.1 入驻资质要求

**基础资质：**
```
✅ 实名认证（身份证）
✅ 手机号绑定
✅ 银行卡绑定
```

**店铺资质：**
```
✅ 营业执照（个体户或公司）
✅ 品牌授权（如销售品牌商品）
✅ 行业资质（特殊类目需要）
✅ 保证金（根据类目 2000-50000 元）
```

**常见类目保证金：**
```
- 服装鞋帽：2000-5000 元
- 美妆护肤：5000-10000 元
- 食品饮料：5000-10000 元
- 数码电器：5000-10000 元
- 珠宝首饰：10000-50000 元
```

### 2.2 直播行为规范

**禁止行为：**
```
❌ 穿着暴露或奇装异服
❌ 抽烟、喝酒、吃东西
❌ 说脏话或低俗内容
❌ 展示危险行为
❌ 长时间离开镜头
❌ 播放录播内容
❌ 引导私下交易
❌ 展示二维码或联系方式
❌ 贬低竞品或平台
❌ 涉及政治、宗教、色情、暴力
```

**直播时长要求：**
```
- 单场直播建议≥1 小时
- 每周建议直播 3-7 场
- 固定时间直播有利于粉丝养成习惯
```

**直播互动规范：**
```
✅ 积极回复评论
✅ 引导点赞和关注
✅ 感谢礼物和打赏
❌ 不回复负面评论（冷处理）
❌ 不与观众争吵
❌ 不承诺做不到的事
```

### 2.3 商品发布规则

**商品标题规范：**
```
✅ 准确描述商品
✅ 包含核心关键词
✅ 字数控制在 30 字以内
❌ 不使用极限词（最好、第一、顶级）
❌ 不虚假宣传（全网最低、100% 有效）
❌ 不蹭热点（与商品无关的明星、品牌）
```

**商品图片规范：**
```
✅ 图片清晰（至少 800×800）
✅ 主图展示商品全貌
✅ 详情图展示细节
✅ 图片与实物一致
❌ 不使用盗图
❌ 不添加水印和联系方式
❌ 不暴露隐私
```

**商品价格规范：**
```
✅ 明码标价
✅ 价格与描述一致
✅ 优惠活动真实有效
❌ 不虚构原价
❌ 不先涨后降
❌ 不低价引流（价格与运费不符）
```

### 2.4 宣传推广规则

**禁用词汇：**
```
【极限词】
最好、最佳、第一、顶级、极品、独家、万能、100%

【功效词】
治疗、治愈、疗效、消炎、抗癌、减肥、丰胸、祛斑

【权威词】
国家级、世界级、XX 领导推荐、XX 机构认证（无证明）

【时间词】
永久、终身、随时、立刻、马上、当天见效
```

**合规表述：**
```
❌ "这是最好的产品"
✅ "这是我很喜欢的产品"

❌ "100% 有效"
✅ "很多人反馈效果不错"

❌ "全网最低价"
✅ "直播间专属优惠价"

❌ "治疗 XX 疾病"
✅ "有助于改善 XX 问题"
```

### 2.5 交易履约规则

**发货时效：**
```
- 一般商品：48 小时内发货
- 预售商品：按约定时间发货
- 定制商品：按约定时间发货
- 超时未发货：赔付订单金额的 30%（最高 500 元）
```

**售后服务：**
```
✅ 7 天无理由退货（部分商品除外）
✅ 质量问题免费退换
✅ 运费险建议购买
✅ 及时处理售后申请
```

**禁止行为：**
```
❌ 拒绝合理售后
❌ 拖延处理时间
❌ 要求买家承担不合理费用
❌ 威胁或骚扰买家
```

### 2.6 违规处罚体系

**扣分规则：**
```
- 轻微违规：扣 1-2 分
- 一般违规：扣 4-6 分
- 严重违规：扣 12 分
- 累计扣分达到 12 分：店铺清退
```

**处罚措施：**
```
【警告】
- 首次轻微违规
- 不影响店铺运营

【限制功能】
- 限制直播权限
- 限制商品发布
- 限制流量推荐
- 期限：7-30 天

【扣除保证金】
- 根据违规程度扣除
- 金额：几百到几万不等

【封店】
- 严重违规或屡教不改
- 冻结资金
- 永久关闭店铺
```

### 3.2 社区规范

**内容规范：**
```
✅ 真实分享使用体验
✅ 标注广告合作（报备笔记）
✅ 图片清晰美观
✅ 文案真诚有用
❌ 不发布虚假内容
❌ 不抄袭搬运
❌ 不发布硬广
❌ 不引导私下交易
❌ 不发布违规内容
```

**直播规范：**
```
✅ 提前预告直播
✅ 准备充分的产品介绍
✅ 积极互动回答问题
✅ 展示真实产品
❌ 不直播违规内容
❌ 不长时间离开

```

- 报备后需标注"广告"或"合作"标识
```

**违规处罚：**
```
- 未报备商业内容：限流、删除笔记
- 多次违规：限制商业权限、封号
```

### 3.4 违规处罚

**笔记违规：**
```
- 删除违规笔记
- 限制笔记曝光
- 扣除信用分
- 限制发布权限
```

**账号违规：**
```
- 警告
- 限制功能
- 封禁账号（7 天/30 天/永久）
```

### 6.2 消费者权益保护

**七天无理由退货：**
```
适用商品：
✅ 大部分商品适用
❌ 定制商品
❌ 鲜活易腐商品
❌ 在线下载或已拆封的音像制品
❌ 已拆封的计算机软件
❌ 交付的报纸期刊
❌ 其他根据商品性质不宜退货的商品

退货条件：
✅ 商品完好
✅ 不影响二次销售
✅ 包装配件齐全
✅ 在 7 天内提出
```

**三包政策：**
```
【包修】
- 保修期内免费维修
- 保修期根据商品类别不同

【包换】
- 质量问题可换货
- 一般 7-15 天内

【包退】
- 质量问题可退货
- 7 天内可无理由退货

### 6.3 知识产权保护

**禁止行为：**
```
❌ 销售假冒商品
❌ 未经授权使用他人商标

✅ 使用自有图片或获得授权
✅ 原创文案
✅ 申请自有专利

### 6.4 个人信息保护

**禁止行为：**
```
❌ 泄露买家个人信息
❌ 未经同意使用买家信息
❌ 出售买家信息
❌ 骚扰买家

**合规做法：**
```
✅ 保护买家隐私
✅ 仅用于订单处理
✅ 按规定期限保存

### 6.5 税务合规

**纳税义务：**
```
✅ 依法纳税
✅ 如实申报收入
✅ 保留交易凭证

## 七、违规风险防范

### 7.1 日常检查清单

**直播前检查：**
```
☐ 检查商品资质是否齐全
☐ 检查商品描述是否合规
☐ 检查价格是否真实
☐ 检查直播间环境
☐ 准备合规话术
☐ 检查设备正常
```

**直播中注意：**
```
☐ 不使用违规词汇
☐ 不展示违规内容
☐ 不引导私下交易
☐ 不贬低竞品
☐ 如实介绍产品
☐ 及时回复问题
```

**直播后复盘：**
```
☐ 检查是否有违规内容
☐ 处理售后问题
☐ 回复用户评论
☐ 记录问题改进
```

### 7.2 违规应对流程

**收到违规通知：**
```
Step 1: 仔细阅读违规通知
        → 了解违规原因和处罚措施

Step 2: 核实违规情况
        → 查看相关记录
        → 确认是否确实违规

Step 3: 如确实违规
        → 立即整改
        → 提交申诉材料（如有必要）
        → 接受处罚

Step 4: 如有异议
        → 准备申诉材料
        → 在规定时间内申诉
        → 等待处理结果

Step 5: 总结经验
        → 记录违规原因
        → 完善合规流程
        → 避免再次违规
```

### 7.3 申诉材料准备

**申诉材料：**
```
- 违规情况说明
- 整改措施说明
- 相关证据材料
- 承诺书
```

**申诉要点：**
```
✅ 态度诚恳
✅ 事实清晰
✅ 证据充分
✅ 整改到位
✅ 承诺不再犯
```
