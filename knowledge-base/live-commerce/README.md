```

```
┌─────────────────────────────────────────────────────────────┐
│                    直播带货核心 SOP 流程                      │

│   └─→ 5. 准备直播设备                                        │
│                                                             │
│  【选品上架】1-2 周                                           │
│   │                                                         │
│   ├─→ 1. 市场调研（数据工具 + 热销榜）                         │
│   ├─→ 2. 寻找供应商（1688/代发平台）                          │
│   ├─→ 3. 采购样品测试质量                                    │
│   ├─→ 4. 采集商品信息并优化                                  │

│   └─→ 6. 加入精选联盟                                        │

│  【内容运营】持续                                            │

│   ├─→ 2. 直播带货（每周 3-7 场）                               │
│   ├─→ 3. 粉丝互动 + 私域运营                                 │
│   └─→ 4. 数据分析 + 持续优化                                 │
│                                                             │
│  【订单处理】持续                                            │
│   │                                                         │
│   ├─→ 1. 客户下单 → 店铺后台                                │
│   ├─→ 2. 供应商平台采购                                      │
│   ├─→ 3. 填写客户地址                                        │
│   ├─→ 4. 回填物流单号                                        │
│   └─→ 5. 客户收货确认                                        │
│                                                             │
│  【售后处理】持续                                            │

│   ├─→ 1. 处理咨询和投诉                                      │
