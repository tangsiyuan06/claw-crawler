### 2.2 核心优势对比

| 维度 | 传统品牌 | 网易严选 | 消费者获益 |
|------|----------|----------|------------|
| 品牌溢价 | 30-50% | 0% | 价格降低 30-50% |
| 营销成本 | 15-25% | 5-10% | 价格降低 10-15% |
| 渠道成本 | 20-30% | 5-10% | 价格降低 15-20% |
| 库存成本 | 10-15% | 5-8% | 价格降低 5-7% |
| **最终价格** | **100%** | **40-60%** | **节省 40-60%** |

### 2.3 供应商合作模式

#### 合作类型

| 类型 | 说明 | 代表品类 | 合作深度 |
|------|------|----------|----------|
| **ODM 合作** | 工厂设计 + 生产，严选贴牌 | 家居、厨具 | 深度绑定 |
| **OEM 合作** | 严选设计，工厂生产 | 服装、箱包 | 中度合作 |
| **品牌授权** | 授权工厂使用严选品牌 | 个护、食品 | 浅度合作 |
| **联合开发** | 共同研发新品 | 科技产品 | 战略绑定 |

```

│                                                                      │
│  ┌──────────────┐         ┌──────────────┐         ┌──────────────┐ │
│  │   消费者     │         │    平台      │         │    工厂      │ │
│  │              │         │              │         │              │ │
│  │ • 拼团下单   │────────▶│ • 需求聚合   │────────▶│ • 按需生产   │ │
│  │ • 低价需求   │         │ • 订单集单   │         │ • 批量生产   │ │
│  │ • 社交分享   │         │ • 数据反馈   │         │ • 成本优化   │ │
│  └──────────────┘         └──────────────┘         └──────────────┘ │
│                                                                      │
│  核心逻辑：                                                          │
│  • 10 万人拼团 = 工厂大批量生产 = 成本降低 50%+                      │

│  • 零库存生产 = 库存成本降低 90%+                                    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

#### 关键数据

| 指标 | 数值 | 说明 |
|------|------|------|

| 价格优势 | 比传统电商低 30-50% | 极致性价比 |
| 工厂合作数 | 1000+ | 新品牌计划工厂 |
| 单品销量 | 10 万 + 常见 | 集单效应明显 |
| 库存周转 | 7-15 天 | 接近零库存 |
