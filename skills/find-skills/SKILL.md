# Find Skills Skill

This skill helps discover and install new OpenClaw skills from various sources.

## Features
- Search skills on ClawHub
- Install skills from GitHub repositories  
- Manage local skill dependencies
- Validate skill structure and compatibility

## Usage
Use this skill when you need to:
- Find new capabilities for your OpenClaw agent
- Install community-created skills
- Update existing skills
- Discover skills for specific use cases

## Installation
This skill should be placed in your workspace/skills/ directory.
